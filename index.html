Exercicio 5: Projecto Calculadora

Neste projecto deve-se contruir uma calculadora baseada em web.

Parte 1.

Faca um programa em JS que efectue operacoes matematicas basicas (Adicao, subtracao e divisao).
A funcao deve ter:

  • Um menu onde o utilizador pode escolher a operacao que deseja efectuar
  • No menu, deve existir uma opcao para sair do programa.

Use:

  • Uma estrutura de decicao e/ou de repeticao adequada para o menu.
  • Funcoes separadas para cada operacao.
  • Os 3 tipos de funcao estudadas nestas aulas.

OBS:

  • Use prompt para mostrar o menu e receber os dados.
  • Use alert para mostrar os resuldados das operacoes

Parte 2.

Desenhe o layout de uma calculadora no index do projecto.

NOTAS:

  1. Cria um novo projecto (separado desta aula) para o programa e use a estrutura de projectos que vimos no inicio das aulas.
  2. Crie um repositorio git para o projecto
  3. Crie um repositorio remoto (no GitHub) e faca o upload do projecto para este repositorio.
  4. Adicione o usuario "Jay-Ubisse" como colabrador do projecto no github.
